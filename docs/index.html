Пушной аукцион

Система управления зверофермами и учёта торгов пушниной

О проекте

Консольное приложение на C++ для автоматизации деятельности пушного аукциона. Система позволяет управлять зверофермами, учитывать выставленную на торги пушнину, фиксировать результаты продаж и формировать финансовую отчетность. Приложение использует базу данных SQLite.
