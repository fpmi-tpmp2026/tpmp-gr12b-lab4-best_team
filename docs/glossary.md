# Глоссарий предметной области «Пушной аукцион»

| Термин | Описание |
| :--- | :--- |
| **Аукцион** | Публичная продажа пушнины, где товар достается покупателю, предложившему наивысшую цену. |
| **Звероферма** | Сельскохозяйственное предприятие, занимающееся разведением пушных зверей (норка, соболь, песец и др.). |
| **Лот (пушнины)** | Единица торга на аукционе. Набор шкурок одного вида меха, одного сорта, выставленный конкретной зверофермой. |
| **Сорт меха** | Качественная характеристика пушнины (высший, первый, второй), влияющая на цену. |
| **Заявленная цена** | Начальная стоимость единицы лота, установленная зверофермой-поставщиком перед началом торгов. |
| **Продажная цена** | Фактическая цена, по которой лот был продан на аукционе (может быть выше или ниже заявленной). |
| **Прибыль фермы** | Финансовый результат, рассчитываемый как разница между продажной и заявленной ценой за все проданные единицы лота. |
| **Категория покупателя** | Тип покупателя на аукционе: меховая фабрика, ателье, частное лицо. |
| **Управление зверосовхозами** | Администратор системы. Пользователь, имеющий полный доступ ко всем операциям (CRUD, фиксация торгов, статистика). |
